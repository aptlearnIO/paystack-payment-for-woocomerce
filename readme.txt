Easily integrate Paystack payment gateway into your WooCommerce store with this plugin by [aptLearn](https://www.aptlearn.io). 

== Description ==

The Paystack Gateway for WooCommerce plugin by [aptLearn](https://www.aptlearn.io) allows you to accept online payments from local and international customers through Paystack, a payment gateway that's easy to set up and free to use. 

This plugin extends WooCommerce, providing an easy and seamless integration with Paystack. With just a few clicks, you can start accepting payments via Paystack on your WooCommerce store.

Key Features:
- Easy setup: Simply enter your Paystack API keys to start accepting payments.
- Live and Test modes: Test your setup with test transactions before going live.
- Transaction fee: Optionally add a transaction fee for Paystack payments.
- Customizable success and failure URLs: Redirect customers to custom URLs upon successful or failed payments.
- Payment status selection: Choose the order status after a successful payment, either 'Processing' or 'Completed'.
- Secure: This plugin communicates securely with Paystack's servers via the official Paystack PHP Library.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/paystack-gateway-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce->Settings->Payments screen to configure the plugin.
4. You can find 'Paystack' as a payment option under WooCommerce -> Settings -> Payments.

== Frequently Asked Questions ==

= Does this plugin work with the latest version of WordPress and WooCommerce? =
Yes, this plugin is compatible with the latest versions of both WordPress and WooCommerce.

= Where can I get my Paystack API keys? =
You can get your API keys from the Paystack settings page in your Paystack account.
